# Fake google keyboard shortcuts.

## Overview

This script working only google.com.
We love keyboard shortcuts,but google disabled that function.
What can we do?
Use this extension:)


## Short cut list.

- j(next link)
- k(previous link)
- Enter(open link)
- Ctrl+Enter(open new tab)
- Shift+Enter(open new tab at background)
